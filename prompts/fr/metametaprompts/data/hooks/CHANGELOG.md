# Hooks Changelog

All notable changes to project hooks.

## [1.0.0] - 2026-02-02

### Added
- Initial hooks configuration
- Session initialization hooks (load-memory, check-skills-sync)
- Guardrails for sensitive files (.env, credentials, secrets)
- Guardrails for destructive operations (rm, force-push, hard-reset, clean)
- Automation hooks (auto-generate, inclusivity-reminder, translation-sync)
- Audit logging hooks (disabled by default)
- Session end hooks (memory-update-reminder)
- Validation hooks with agents (disabled by default)
